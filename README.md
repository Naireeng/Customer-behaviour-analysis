# Customer-behaviour-analysis
📊 Data Analytics Project
Overview

This project demonstrates an end-to-end Data Analytics workflow, from data cleaning to business insights. It includes data preprocessing in Python, SQL analysis, an interactive Power BI dashboard, and a final business report and presentation.

Dataset

The dataset contains business-related data used to analyze trends, customer behavior, product performance, and key business metrics.

Tools & Technologies
Python: Pandas, NumPy, Matplotlib, Seaborn
SQL: PostgreSQL / MySQL / SQL Server
Power BI: Data Visualization & Dashboard
Gamma: Presentation
Microsoft Word: Business Report

Project Workflow
Load and inspect the dataset in Python.
Perform Exploratory Data Analysis (EDA).
Clean and preprocess the data.
Analyze the data using SQL queries.
Build an interactive Power BI dashboard.
Summarize insights in a report and presentation.

Dashboard Highlights
📈 Sales & Revenue Analysis
👥 Customer Insights
🛍️ Product Performance
📊 Key Performance Indicators (KPIs)
📅 Trend Analysis

How to Run
Clone the repository.
Install the required Python libraries.
Run the Jupyter Notebook.
Execute the SQL queries.
Open the .pbix file in Power BI Desktop.
